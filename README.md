# PHP Class Template

Note: This is a playground repository for PHP and Bash. Nothing too serious here.

Sample usage:
```
$ bash script.sh <dir1> [<dir2> ...]
```

Sample call:
```
$ php script.php <resource>
```

Sample resource (JSON):
```
{
    "namespace": "App\\Common",
    "className": "Animal",
    "fields": [
        {
            "name": "name",
            "type": "string",
            "docs": "The animal's name."
        },
        {
            ...
        }
    ]
}
```
